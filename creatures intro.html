Lore Page

Unlike the Persona section, this section contains Lore Pages. This means that there are special pieces of information for each type of creature. Every Lore page is laid out with this information:

This is the Creature Type. This is learned by Identifying the creature.

This is the Description. This can be used by the Host to describe what the creature looks, smells, sounds, or feels like when it is encountered.

These are the Lore Items. These are learned by characters if they have a high enough Passive Lore or they roll a high enough Analyze using the correct Lore. See more information about this below.

This is the Flavor Quote. This is just for flavor! :)

These are the Story Details. These facts are typically known by anyone that can identify the creature, but not always. These facts are to help the Host integrate this creature meaningfully into the larger story.

This Lore may be learned for free if the character has a high enough passive Art Total in the Lore and/or Lores listed here (no temporary buffs are included in this calculation).

If the character does not have a high enough passive Lore, they must attempt to Analyze the target. They may only roll Analyze (to learn Lore) once per day per target. Make it count! The Host always lets the character know which Lore they must use if the character has not yet identified ther target. (See Advice below).

Identifying a Target

On each Lore page, there is an asterisk placed next to one Passive Lore. This is used to identify that creature.

It costs nothing to identify a creature. Identifying grants the character two pieces of information:

1. What type of creature it is (dragon, groll, etc)

2. What Lore type is required to Analyze and use spells on that creature

The type of creature as well as any Lore learned can be shared with others. But the Lore Type of a creature can only be learned by a character identifying that creature themselves.

For Nature’s Domain creatures (except Wyvern), the Passive Lore required is simply [20 less] than whatever is listed for that creature or [0] (whichever is lower).

Advice for Tricky Lore Situations

What if a wizard wants to portal a spell on a Vampire, but their character doesn’t actually know that their target is a Vampire yet? The correct strategy is this:

If the character has not identified the creature yet, the Host must suggest to the player which Lore to roll based on the following (loose) taxonomies (in order with examples):

- Does it appear as though it has obviously been created or is non-sentient? -> Alchemy.

- Else, does it appear as though it is from another plane of existence or is non-physical? -> Essence

- Else, does it appear to be non-human in any way? -> Nature.

- Else -> Common

Take note though: if you suggest the incorrect Lore to a player, they receive [-50] to their maneuver. So you should be consistent about which creatures could be “tricky” to guess.

(Groll, incorrect): “This appears to be an ordinary tree. Roll Alchemy Lore.”

(Spectre, correct but lucky) “They appear to have no corporeal form. Roll Essence Lore.”

(Scudgie, incorrect) “They appear sentient and have a non-human form. Roll Nature Lore.”

(Vampire, incorrect) “They appear to be an ordinary human. Roll Common Lore.”
